背景

初步了解Appium各个功能之后，应该把这些功能进行一些封装，否则整个代码会比较难看，可用性和重用性也会很差。本文是我这段时间使用Appium的一些想法，仅供参考。

操作系统：Mac OS X EI Caption

Appium: 1.4.16

Java: java version “1.7.0_79”

node.js: v5.3.0

npm: 3.3.12

手机：小米NOTE4

待测应用： 微证券
