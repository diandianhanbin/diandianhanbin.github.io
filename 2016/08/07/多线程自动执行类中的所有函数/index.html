多线程自动执行类中的所有函数

发表于 2016-08-07   |   分类于 Python   |  

背景

电脑每次开机总有许多服务要起，比如MongoDB、爬虫、本地网站的服务等等，这些都分布在各个文件夹，每次重启电脑都需要到各个目录下去启动服务，虽然也不会花多少时间，但是非常麻烦，所以就看了一下多进程来解决这个问题。

思路

每一个服务都是独立的，切启动之后不会关闭，那么传统的直线型写代码肯定不行，必须要多个进程来处理这些东西。
